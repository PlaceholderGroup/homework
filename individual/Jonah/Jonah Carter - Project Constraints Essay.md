**Security:**  
The project heavily involves the user's personal information. That information will be shared, with user permission, to other users. We need to make sure we take steps to prevent the leaking of user information and/or hacking attempts. 

**Social:**  
By using the application, interaction between people in many different roles is necessary. For example, the interaction between two business partners will differ from the interaction between friends sharing contacts. We should take into consideration the different social contexts in which the multiple uses of our application exist.

**Diversity/Cultural:**  
We want our app to be usable by all; this means that we need to consider things like using colors that are accessible to everyone, no matter their vision level, as well as ensuring we have support for screen readers and other assistive technologies. We will also need to consider when storing and sharing contacts the cultural differences in things like titles, names, etc. Although we may not have the time/knowledge to provide translations ourselves we should consider how translations and other localizations would work and build our app in a way that allows for us to easily support them in the future.

**Economic:**  
We have no funds to work with, but anticipate minimal to no expenses. Our only potential expenses are purchasing RFID cards (which are pretty cheap) and possibly minimal server and/or database expenses. We have some existing hardware that we will prioritize using as well as several free options we can take advantage of if the need arises. We are aware of and willing to subsidize any potential minimal expenses.
