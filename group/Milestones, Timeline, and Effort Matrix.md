## Milestones

1. Complete the sketching  
    - We will sketch an interface to quickly prototype and test different designs. Since we don’t have as much detail at this stage it makes it a lot easier to make and test rapid changes.  
2. Complete the design  
    - Eventually we have to actually make the sketch a nice looking prototype. Completing this milestone will involve translating our sketch into an actual complete design in a tool like Figma.  
3. Complete the frontend  
    - Once we have the design we can start developing the frontend code. This can be worked on independently of the backend and database although some care will need to be taken to make sure both teams are on the same page.  
4. Complete the backend  
    - This is the API that the frontend will interface with. It can be developed independently of the frontend but should be tightly integrated with the database.  
5. Complete the database design  
    - Design the database schema. This will need to be tightly integrated with the backend, and some consideration should be taken to ensure it works well with the frontend.  
    - Work on making the database efficient for use.  
6. Testing the application  
    - We will test the application on both iOS and Android  
    - We will create unit tests and integration tests
